## Getting Started

No install or build step required вҖ” pure HTML, CSS, and JavaScript.

```bash
git clone git@github.com:FARHANMOHAMMED-R/farhan_new.git
cd farhan_new
open index.html
```

Or on Windows:
```bash
start index.html
```

### Optional: run with a local server

```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve .
```

Then visit `http://localhost:8000`
